=======
BeeWare
=======

**Languages**:
  `Englisch <../index.html>`__

**Einmal schreiben. Überall verwenden**

BeeWare ist ein Tool- und Bibliothekenpacket, mit welchem man Cross-Platform
Programme schreiben kann. Es enthält:

* `Toga <https://toga.readthedocs.io>`__, ein Cross-Platform, Widget Toolkit;
* `Briefcase <https://briefcase.readthedocs.io>`__, ein Werkzeug zum verpacken von Python
  projekten als vollwertige Installationsdateien;
* `Rubicon ObjC <https://rubicon-objc.readthedocs.io>`__, eine Bibliothek, mit der man mit
  Objective C Bibliotheken für iOS and macOS in Python arbeiten kann.;
* `Rubicon Java <https://rubicon-java.readthedocs.io>`__, eine Bibliothek, mit der man mit
  Java libraries in Python arbeiten kann.;
* Python installationen für Platformen, für die es keine offiziellen Python-binaries gibt.

Diese Werkzeuge können alleine oder in Verbindung miteinander verwendet werden.

Die BeeWare suite enthält auch `software development tools
<https://beeware.org/project/projects/tools/>`__ und `apps
<https://beeware.org/project/projects/applications/>`__ welche mit der BeeWare
suite geschrieben wurden.

Beeware unterstützt macOS, Windows, Linux(unter verwendung von GTK); Android
und iOS Handys; und platforms such as Android and iOS; und Django(Internet).
Weitere Platformen, wie Set-Top-Boxen und Smartwatches sind in Planung.

.. admonition:: Platform support

    BeeWare ist work in progress. Manche Werkzeuge und Bibliotheken sind weiter,
    und manche Platformen funktionieren schon besser. Web und Android noch in Erprobung.

Sie wollen BeeWare ausprobieren? :doc:`Erstellen wir eine Cross-Platform-App
in Python! <tutorial/tutorial-0>`
